### Filters

| Field name          | description                                         |
| ------------------- | --------------------------------------------------- |
| **name**            | Filters name                                        |
| **url**             | RSS or GET url                                      |
| **type**            | `ss` / `city24`                                     |
| **last_entry_time** | Time of he last advertisement (to avoid duplicates) |
| **last_id**         | Last sent advertisements ID (to avoid duplicates)   |
| **email**           | Email to which results should be sent               |
