### Filters

| Field name               | description                                  |
| ------------------------ | -------------------------------------------- |
| **name**                 | Filters name                                 |
| **url**                  | RSS or GET url                               |
| **type**                 | `ss` / `city24`                              |
| **last_build_timestamp** | Time of he last filter (to avoid duplicates) |
| **email**                | Email to which results should be sent        |
